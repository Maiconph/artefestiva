Gateway de Pagamento (Asaas)

Configure a chave de API e o Webhook de retorno para PIX.

Chave gerada no painel do Asaas em Configurações > Integração > Chave de API.

Configuração do Webhook

Copie a URL abaixo e cole no painel do Asaas em Configurações > Integrações > Webhooks para Cobranças. Isso permite que o sistema saiba imediatamente quando a cliente pagar o PIX.

Eventos a serem marcados no Asaas:
PAYMENT_RECEIVED PAYMENT_CONFIRMED
